FoxAir / Phnix Control V0.2.19

Neues Feature: Wissensdatenbank / Beschreibungen zu Datenpunkten

- Parameter-Einstellungen zeigt jetzt ein Beschreibungsfeld unter der Tabelle.
- Mouse-Over oder Auswahl einer Zeile zeigt Zusatzinfos, wenn sie im Mapping vorhanden sind.
- Tooltips der Tabellenzellen enthalten ebenfalls die Beschreibung.
- Unterstuetzte JSON-Felder pro Register:
  - description: kurze Beschreibung
  - knowledge oder explanation: laengere Erklaerung / Zusammenhang
  - notes oder hint: Notizen / Hinweise
  - default: Defaultwert
  - source: Quelle
  - source_app_video: App-Video-Quelle

Beispiel im Mapping:
{
  "1344": {
    "name": "A40 / Nenn-Wasserdurchfluss",
    "type": "FLOW_M3H_X100",
    "description": "Rated Water Flow / Nenn-Wasserdurchfluss",
    "knowledge": "Die App zeigt raw/100; raw 50 = 0,5 m3/h.",
    "source": "App-Video / Projektanalyse"
  }
}
