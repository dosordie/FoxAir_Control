Analyse DEMONS_V1.3.ASM für Warmlink GUI v20
============================================

Quelle: /mnt/data/DEMONS_V1.3.ASM
Ziel: Nur zusätzliche Erkenntnisse übernehmen, bekannte Namen nicht unnötig überschreiben.

1) Legacy-Parameterpakete / Init-Lesen
--------------------------------------
Die ältere V1.3 verwendet andere Parameterpaket-Startadressen als die neuere Firmware:

- Paket 1: 0x03FA / 1018, Länge 73 Register, Kommentar: 1018--1090
- Paket 2: 0x044D / 1101, Länge 80 Register, Kommentar: 1101--1180
- Paket 3: 0x04A7 / 1191, Länge 80 Register, Kommentar: 1191--1270

Zusätzlich tauchen optionale/testweise Blöcke auf:
- 0x05B5 / 1461
- 0x060F / 1551
Diese sind in V1.3 kommentiert/teilweise deaktiviert und werden nicht automatisch gelesen.

Umsetzung v20:
- Checkbox "V1.3" neben Init-Blöcke ergänzt.
- Diese liest nur die drei klar aktiven Legacy-Blöcke 1018/73, 1101/80, 1191/80.
- Parser-Wortblockliste erweitert um 0x03FA, 0x044D, 0x04A7, 0x05B5, 0x060F.

3) Modus-/Status-Value-Maps
---------------------------
2012 / 0x07DC:
- 2 = Abtauen
- 3 = Sterilisieren / Desinfektion
Die V1.3 blendet bei diesen Werten den normalen Betriebsmodus aus und zeigt stattdessen den Sonderstatus.

1059 / F01 Lüftertyp, Legacy-Hinweise:
- 0 = Hochgeschwindigkeits-Lüfter
- 1 = zweistufiger Lüfter
- 3 = DC-Lüfter
- 4 = DC-Lüfter externer Antrieb (aus neuerem Mapping)

4) Fehlerregister / Bitfelder aus Fault_Status_Display
------------------------------------------------------
Neu im Core: generischer bit_map/FAULT_BITS Decoder.
Wenn ein Fehlerregister einen gesetzten Bitwert hat, zeigt die Tabelle die bekannten Bits direkt an.

2085 / Fehler 1:
- Bit14 = Raumtemperaturfühler / Room Temp Fehler

2089 / Fehler 5:
- Bit0  = T01 Einlasswassertemperatur Fehler
- Bit1  = T02 Auslasswassertemperatur Fehler
- Bit2  = System 1 Coil-/Verdampfertemperatur 1 Fehler
- Bit3  = Umgebungstemperatur / AT Fehler
- Bit4  = System 1 Rückgas-/Sauggastemperatur Fehler
- Bit5  = System 1 Frostschutztemperatur 1 Fehler
- Bit6  = System 1 Frostschutztemperatur 2 Fehler
- Bit9  = System 1 EVI-/Enthalpie Eintrittstemperatur Fehler
- Bit10 = System 1 EVI-/Enthalpie Austrittstemperatur Fehler
- Bit11 = System 1 Abgastemperatur Fehler
- Bit13 = System 1 Drucksensor Fehler

2090 / Fehler 6:
- Bit0  = System 2 Coil-/Verdampfertemperatur Fehler
- Bit1  = System 2 Rückgas-/Sauggastemperatur Fehler
- Bit2  = System 2 Frostschutztemperatur 1 Fehler
- Bit3  = System 2 Frostschutztemperatur 2 Fehler
- Bit6  = System 2 Abgastemperatur Fehler
- Bit7  = System 2 Drucksensor Fehler
- Bit8  = Wassertanktemperatur Fehler
- Bit9  = System 2 EVI-/Enthalpie Eintrittstemperatur Fehler
- Bit10 = System 2 EVI-/Enthalpie Austrittstemperatur Fehler
- Bit11 = DC-Lüfter 1 Fehler
- Bit12 = DC-Lüfter 2 Fehler
- Bit13 = E082 Fehler, genaue Bedeutung offen
- Bit15 = E081 Fehler, genaue Bedeutung offen

5) Nicht blind übernommen
-------------------------
- 0x05B5/0x060F sind als Legacy-Wortblock-Typen bekannt, aber nicht in der Init-Checkbox V1.3 enthalten, weil sie in V1.3 kommentiert/teilweise widersprüchlich beschriftet sind.
- 1074/F10 Lüfteranzahl hat in V1.3 andere Wertekommentare als im aktuellen Mapping; deshalb nicht überschrieben.
- Bestehende konkrete Handbuch-/Excel-Namen wurden nicht durch ältere ASM-Kommentare ersetzt.
